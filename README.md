## Installation

### Prerequisites

Install the dependencies

```R
install.packages("jsonlite")
install.packages("httr")
install.packages("caTools")
```

### Build the package

```sh
git clone https://github.com/zhang-ncbi/dataset_r_test
cd dataset_r_test
R CMD build .
R CMD check ncbi.datasets_0.1.0.tar.gz
R CMD INSTALL ncbi.datasets_0.1.0.tar.gz
```

### Install the package

```R
install.packages("ncbi.datasets")
```

To install directly from Github, use `devtools`:
```R
install.packages("devtools")
library(devtools)
install_github("zhang-ncbi/dataset_r_test")
```
